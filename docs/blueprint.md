# **App Name**: Gusto Navigation

## Core Features:

- Fixed Navigation Bar: Create a fixed navigation bar at the top of the page that persists as the user scrolls.
- Logo: Display 'Gusto' text logo on the left side of the navigation bar using Playfair Display font.
- Navigation Links: Create 'Home', 'Menu', 'About', and 'Contact' links in the center of the navigation bar.
- Log In Button: Implement a 'Log In' button with a transparent background and bordered style on the right side.
- Reserve a Table Button: Implement a 'Reserve a Table' button with a gradient fill on the right side.
- On Scroll Effect: Make the navigation bar slightly translucent when the user scrolls down.

## Style Guidelines:

- Primary Gradient: #667eea to #764ba2 (calm, trustworthy blue to sophisticated purple).
- Secondary Gradient: #f093fb to #f5576c (warm, energetic pink to coral) for button accents and highlights.
- Neutral Background: #f9f9f9 (very light off-white).
- Surface Color: #ffffff (pure white for content cards and navbars).
- Primary Text: #333333 (almost black for high readability).
- Secondary Text: #666666 (medium gray for less important text).
- Headings: 'Playfair' serif for elegance and sophistication.
- Body Text: 'Lato' sans-serif for clean, modern readability.
- Border-Radius: 8px for buttons. 0px for input fields.
- Subtle shadow: 0 4px 12px rgba(0, 0, 0, 0.1) for cards and hover states.
